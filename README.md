# 🌤️ Weather App

🔗 **Live Demo:** https://abhinavw12.github.io/Weather-App/

A simple and responsive Weather Application built using HTML, CSS, and JavaScript that provides real-time weather information for any city using a weather API.

---

## 📌 Features

✅ Search weather by city name

✅ Real-time weather updates

✅ Displays temperature in Celsius

✅ Shows humidity percentage

✅ Shows wind speed

✅ Dynamic weather icons

✅ Responsive and modern UI

✅ API integration for live weather data

---

## 🚀 How It Works

1. Enter a city name in the search box.
2. Click the search button.
3. The application fetches live weather data from the API.
4. Weather details such as temperature, humidity, and wind speed are displayed instantly.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/abhinavw12/Weather-App.git
```

### Open the Project Folder

```bash
cd Weather-App
```

### Run the Project

Simply open `index.html` in your browser.

---

## 🔑 API Configuration

1. Get your API key from your weather service provider.
2. Open the JavaScript file.
3. Replace the existing API key with your own.

```javascript
const apiKey = "YOUR_API_KEY";
```
